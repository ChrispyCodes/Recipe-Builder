## Description

_*This is a full stack Recipe Blog application with full CRUD capabilities utilizing a React front end and a C# backend*_

# Technologies Used:

1. Internet browser
2. VS Code
3. React
4. .Net Framework
5. Entity Framework
6. React-Bootstrap
7. Material-UI
